--------------------------------<<<<<<<<<<<<<INSTALACIONES>>>>>>>>>>>>>>>----------------------------

El proyecto está desarrollado en Python, Html, CSS y JavaScript.

Es necesaria la instalación de FireBase_admin y Flask y un IDE que soporte python para lanzar el proyecto.


-----------------------------------<<<<<<<<<<<<<TEMATICA>>>>>>>>>>>>>>>-------------------------------

Se ha desarrollado un proyecto con temática Final Fantasy VII

-------------------------------<<<<<<<<<<<<<FUNCIONALIDADES>>>>>>>>>>>>>>>---------------------------

Se ha desarrollado una página web, utilizando flask y FireBase usando BBDD no relacional.

La página web está desarrollada para hacer un inventario de materias(magias), y armas.

Cada materia tiene sus carácterísticas, al igual que cada arma, el intuitivo y vistoso menú guiará al 
usuario de una manera fácil a cada una de las funciones.

Las funciones que cumple la web es un crud, tanto de materias como de armas.

-------------------------------<<<<<<<<<<<<<FUNCIONES EXTRAS>>>>>>>>>>>>>>>---------------------------

También se han añadido dos opciones extras en el menú principal.

1ª Gestión de BBDD, donde se permite al usuario, borrar la base de datos entera y otra opción en la que se meten 
10 ejemplos de materias y 10 ejemplos de armas.

2ª Se pensó en implementar personajes, donde podrían enlazarse las armas, pero por falta de tiempo, no se ha 
implementado completamente, solo está la parte visual, sin ningún efecto sobre la bbdd.
